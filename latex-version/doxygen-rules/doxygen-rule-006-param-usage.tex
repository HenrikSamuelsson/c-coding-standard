\subsection*{\doxygenRule{}}

A Doxygen comment block for a function shall use the \textbackslash param command for each function parameter also include the direction attribute as well as an description.

\subsubsection*{Examples}

\noindent
\begin{minipage}[t]{0.48\textwidth}
    \lstinputlisting[
        xleftmargin = 3.4pt,
        firstline = 5,
        lastline = 7,
        title = {Non-Compliant}
    ]{../code/doxy-rule-001.c}

\end{minipage}\hfill
\begin{minipage}[t]{0.48\textwidth}
    \lstinputlisting[
        xrightmargin = 3.4pt,
        firstline = 2,
        lastline = 20,
        title = {Compliant}
    ]{../code/doxy-rules/doxy-rule-006-compliant.c}
\end{minipage}
